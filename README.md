# party

Vendored C dependency tarballs.

## Usage

Use the `latest` URLs below in `build.zig.zon` dependency declarations.
These URLs are stable and only change when a library version is updated.

To pin to a specific snapshot, replace `latest` in the URL with a date tag (e.g. `2026-03-05`).

## Adding or updating a tarball

1. Place `.tar.xz` files in the repo root
2. Run `make release`

This creates a dated snapshot release and updates the `latest` release.
